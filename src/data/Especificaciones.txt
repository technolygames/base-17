-Para que pueda mostrar notificaciones, es necesario tener instalado sistemas operativos como: Windows 10, GNOME o KDE. No soporta versiones anteriores de Windows (ej.: Windows XP, Vista, 7, 8-8.1)
Formulario:
-Para introducir una imagen, selecciona alguno de los formatos disponibles.
